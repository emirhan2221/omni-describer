## 🚀 How to use Omni Describer

Open the app using the shortcut on your desktop. The interface presents a simple menu.

### Working with images
You can drop images into the app window. The AI analyzes the picture. It generates text that describes the scene. Your screen reader speaks this description immediately.

### Working with videos
Load a video file into the specific video tab. The app watches the frames and listens to the audio. It creates descriptions for important visual actions. These descriptions sync with the video timeline.

### Working with PDF files
Import your documents into the PDF section. The software scans the pages. It converts images and charts within your documents into plain text. This makes long files much easier to navigate.

Do I need an internet connection?
Yes, the AI processes data on remote servers. A stable connection ensures fast feedback.

Does the app store my files?
No, the tool processes your files in real time. It discards them after the description session ends.

Can I change the voice settings?
Yes, the app uses your Windows system voice settings. You can modify these in the Windows Control Panel to change speed or pitch.

Is it secure?
The app uses encrypted connections. Your data stays private during transfers.

What file types does the app accept?
You can use standard formats like JPG, PNG, MP4, and PDF.
